# Tools Used in Security Assessment

## 1. OWASP ZAP (Zed Attack Proxy)
- Purpose: Automated web application vulnerability scanning
- Usage:
  - Passive scanning for security misconfigurations
  - Active scanning for injection and client-side vulnerabilities
- Version Used: OWASP ZAP v2.16.1

## 2. Browser Developer Tools
- Purpose: Manual analysis and testing
- Usage:
  - Inspecting HTML elements and JavaScript behavior
  - Observing network requests and responses
  - Injecting test payloads into input fields

## 3. Manual Payload Testing
- XSS payloads (e.g., `<script>alert('XSS')</script>`)
- Basic SQL Injection test strings
- Used to validate scanner findings and application behavior

## 4. Documentation Tools
- PDF format used for reporting
- Markdown used for GitHub documentation

All tools were used in a non-destructive and ethical manner.
